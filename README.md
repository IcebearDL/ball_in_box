## problem description

    二维平面上,有一个(-1,1) (1,-1)的正方形,与m个坐标(mxi,myi)
    已知有n个可变半径的 无边界圆(或者"空") 可供放置在正方形内,但不能交叠,同时不能包含上面的m个坐标;
    求所有圆的最大面积
